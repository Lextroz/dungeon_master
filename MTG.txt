1) Общее меню счётчика
- Изменение количества игроков (2 или 4)
- Включение/выключение счётчика Infect
- Звук окончания таймера

2) Экран счётчика
- Альбомная раскладка
- Блоки с очками здоровья для каждого игрока
	- Экран разделён на 2 или 4 прямоугольника (количество игроков)
	- В каждом блоке есть:
		- Очки здоровья (большими цифрами, начальное значение = 20)
			- При нажатии на число, значение уменьшается на 1
			- Над числом кнопки для увеличения значения на 1 или 5 и для сброса значения
		- Очки Infect (если включено) (рядом со здоровьем, начальное значение = 0)
			- При нажатии на число, значение увеличивается на 1
			- Над числом кнопка для сброса значения
		- Кнопка для вызова дополнительного меню игрока:
			Дополнительное меню содержит:
			- Сыграно карт (Storm counter) (начальное значение = 0) (при нажатии на число значение увеличивается) (имеет кнопку сброса значения)
			- Количество маны (по одному счётчику на каждый цвет (W - белый, B - чёрный, U - синий, G - зелёный, R - красный, С - бесцветный)) (может изменяться в +-1) (начальное значение = 0) (каждая мана имеет кнопку сброса значения)
- Таймер посередине экрана на пересечении блоков 
	- Начальное время таймера = 50 минут (обратный отсчёт)
	- Кнопка сброса времени
	- Таймер запускается при нажатии на него
	- Приостановка таймера опциональна, т.к. между партиями в одной игре таймер не останавливается
